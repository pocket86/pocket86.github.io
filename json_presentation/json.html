JSON Presentation

JSON is simply a way to format data to make it simple for humans and machines to create, exchange, and consume data.

Although a JSON object can be created within a .html page or .js file, best practice teaches us to create stand-alone files with a .json extension.

JSON uses key-value pairs. This binds each value to its key for easy parsing and lookup. This is in contrast to an array, witch uses an index based on position in the array - JSON objects are concidered 'un-ordered' for this reason.

This simple example shows what an object representing a person might look like. It also shows use of the different data types that JSON objects can support, including strings, numbers, booleans, and objects.
